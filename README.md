# NexaScience Corporate Website

株式会社NexaScienceのコーポレートサイトです。

**URL:** https://nexascience.github.io/

## ページ構成

| ページ | ファイル | 内容 |
|---|---|---|
| HOME | `index.html` | ヒーロー、MISSION、VISION、PRODUCT、MEMBERS、COMPANY |
| Lnar | `aixs.html` | Lnar（Linkage for Agent Resources）プロダクト紹介 |
| NEWS | `news.html` | ニュース・プレスリリース |
| CONTACT | `contact.html` | お問い合わせ |

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- GitHub Pages でホスティング
- レスポンシブデザイン（モバイル対応）

## ローカルでの確認

```bash
# 任意のローカルサーバーで起動
python3 -m http.server 8000
# http://localhost:8000 でアクセス
```

## デプロイ

`main` ブランチに push すると GitHub Pages に自動デプロイされます。
